# 📝 AH Digital Blog

**🇧🇷** Blog sobre tecnologia, IA, automação e produtividade para pequenas empresas — criado com Next.js, Supabase e deploy na Vercel.

**🇬🇧** Blog about technology, AI, automation, and productivity for small businesses — built with Next.js, Supabase, and deployed on Vercel.

---

## 🚀 Sobre / About

**🇧🇷** O AH Digital Blog nasceu para compartilhar conhecimento sobre tecnologia aplicada a pequenos negócios. Aqui você encontra artigos sobre IA, automação, ferramentas digitais e como usar tecnologia para resolver problemas reais.
O blog é todo automatizado pelo meu agent hermes, ele pega as noticias principais do dia, reescreve baseado no prompt que criei, posta, faz o git e sobe.
O intuito foi criar essa automação para aprender e ver como funciona a IA no dia a dia.

**🇬🇧** AH Digital Blog was created to share knowledge about technology applied to small businesses. Here you'll find articles about AI, automation, digital tools, and how to use tech to solve real problems.
The blog is fully automated by my agent, Hermes; it grabs the day's top news stories, rewrites them based on a prompt I created, posts them, commits the changes to Git, and deploys them.
The goal was to build this automation to learn and see how AI works in day-to-day life.

🔗 **Live site:** [ah-digital-blog.vercel.app](https://ah-digital-blog.vercel.app)
